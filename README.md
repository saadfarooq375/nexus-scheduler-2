# Nexus AI Scheduler

An intelligent scheduling app powered by Claude AI.

## Setup

1. Install dependencies: `npm install`
2. Run locally: `npm run dev`
3. Build for production: `npm run build`

## Environment Variables

For full AI functionality, add your Anthropic API key in Vercel:
- `VITE_ANTHROPIC_API_KEY` — Get one at console.anthropic.com
